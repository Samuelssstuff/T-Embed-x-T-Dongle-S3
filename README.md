# T Embed x T Dongle - VERSION 1.0

**T-Embed CC1101+ Remote Payload & Live Keyboard** for Bruce JS Interpreter + SwissArmyKnife on T-Dongle S3.

---

### Installation Guide

1. Install **[USBSwissArmyKnife](https://github.com/Nikorasu-07/ds-scripts-USB-Army-Knife)** on your LilyGo T-Dongle S3.  
   Recommended YouTube tutorial: [Watch here](https://www.youtube.com/watch?v=TxhNiqnNuPE&t=319s)

2. Drag the `.ds` files into your T-Dongle’s MicroSD card.

3. Install the latest version of **[Bruce](https://bruce.computer/)** on your T-Embed.  
   Recommended YouTube tutorial: [Watch here](https://www.youtube.com/watch?v=xu8ZPNcjuuM)

4. Plug your MicroSD card into your PC/Laptop.

5. Copy the `TEmbedxTDongle.js` file into the **root** of the MicroSD card.

6. Make sure your T-Dongle’s WiFi SSID and password match (default: `iPhone14` / `password`). If not, edit the `TEmbedxTDongle.js` file accordingly.

7. Plug the MicroSD card back into your T-Embed.

8. Scroll to **"Js Interpreter"**.

9. Click **"load..."**

10. Select **SD Card**.

11. Scroll down and select **`TEmbedxTDongle.js`**

12. Ensure your T-Dongle is connected by clicking **"linux"**.

13. Click **"run linux.ds"**, if the T-Dongle flashes red briefly, the connection is successful.

14. Click the back button to return to the main menu.

15. Go to **"Settings"** to customize options (color scheme, apply motion blur etc.).

**✅ You have successfully connected your T-Embed to your T-Dongle!**
